# Differenciator

Консольный **дифференциатор математических выражений** на C/C++, который:

1. читает выражение из файла;
2. строит **дерево выражения**;
3. рекурсивно строит дерево производной;
4. упрощает результат;
5. генерирует **LaTeX** и итоговый **PDF** с формулой и производной;
6. умеет строить граф дерева через Graphviz.

В проекте нет нормального README, поэтому ниже собрана документация по реальному коду репозитория.

## Что умеет проект

- парсить арифметические выражения в дерево;
- брать производную по переменной `x`;
- поддерживать базовые элементарные функции;
- выполнять упрощение дерева после дифференцирования;
- экспортировать результат в LaTeX/PDF;
- выводить дерево в Graphviz.

## Поддерживаемые операции

Из кода парсера и дифференциатора видно, что проект работает со следующими операциями и функциями:

- `+`
- `-`
- `*`
- `/`
- `^`
- `sin(...)`
- `cos(...)`
- `tg(...)`
- `ctg(...)`
- `log(a,b)`
- `ln(...)`
- `exp(...)`

Также поддерживаются:

- целые числа;
- переменная `x`.

## Логика работы

Точка входа минимальная: `main.cpp` вызывает `Run()`.

Внутри `Run()` происходит:

1. чтение исходного дерева выражения;
2. построение дерева производной;
3. упрощение производной;
4. Graphviz-дамп дерева производной;
5. генерация LaTeX и запуск `pdflatex`.

## Правила дифференцирования, реализованные в коде

В проекте вручную зашиты основные правила:

- `d(c)/dx = 0`
- `d(x)/dx = 1`
- `(f + g)' = f' + g'`
- `(f - g)' = f' - g'`
- `(f * g)' = f' * g + f * g'`
- `(f / g)' = (f' * g - f * g') / (g * g)`
- `(f^g)' = f^g * ( g' * ln(f) + (g / f) * f' )`
- `(sin f)' = cos(f) * f'`
- `(cos f)' = -1 * sin(f) * f'`
- `(tg f)' = (1 / cos(f)^2) * f'`
- `(ctg f)' = (-1 / sin(f)^2) * f'`
- `(log(a, b))' = (1 / (b * ln(a))) * b'`
- `(ln f)' = (1 / f) * f'`
- `(exp f)' = exp(f) * f'`

## Упрощение после дифференцирования

После построения дерева производной проект делает несколько проходов упрощения. В коде явно реализованы:

- свёртка константных поддеревьев;
- удаление умножения и деления на `1`;
- удаление сложения и вычитания с `0`;
- упрощение умножения на `0`;
- упрощение `a^1 -> a`;
- упрощение `a^0 -> 1`.

Это делает результат заметно чище, чем “сырая” производная.

## Формат входа

Парсер ожидает строку формулы в специальном формате.

### Важные особенности

- выражение должно быть **обрамлено символами `$...$`**;
- переменная поддерживается только одна — **`x`**;
- числа читаются как **целые**;
- функции пишутся в явном текстовом виде: `sin(...)`, `ln(...)`, `log(a,b)` и т.д.

### Пример допустимого ввода

```text
$x^2+sin(x)$
```

## Сборка

Из корня репозитория:

```bash
make
```

Запуск:

```bash
make run
```

Очистка:

```bash
make clean
```

Локальный `Differenc/Makefile` собирает проект через `g++` и использует debug-конфигурацию с санитайзерами.

## Зависимости

Минимально по коду нужны:

- `g++`
- `make`
- `Graphviz`
- `pdflatex`

## Выходные файлы

После работы проекта формируются:

- бинарник `Differenciator`;

- LaTeX-файл;
- итоговый PDF с формулой и производной.

В коде PDF и `.tex` создаются через `TexOutput()` и затем компилируются командой `pdflatex`.
